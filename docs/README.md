# 프로젝트 문서 가이드 (Project Documentation Guide)

이 디렉토리는 프로젝트의 모든 기술 문서, 설계안, 그리고 현재 진행 상황을 체계적으로 관리하기 위한 공간입니다.

## 📂 디렉토리 구조 및 역할

### 1. [specs/](./specs/) (요구사항 및 명세)

- **역할:** 프로젝트의 비즈니스 요구사항 및 API 상세 명세를 관리합니다.
- **주요 파일:**
  - `api/`: Swagger/OpenAPI 관련 파일 및 API 상세 명세.
  - `stepN-topic.md`: 각 마일스톤별 기능 요구사항 정의서.

### 2. [guides/](./guides/) (개발 가이드)

- **역할:** 개발 환경 설정, 툴 사용법 등 협업에 필요한 전반적인 가이드를 제공합니다.
