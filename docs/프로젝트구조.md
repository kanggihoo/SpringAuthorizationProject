핵심 구성:

JWT 기반 인증/인가 - Access Token + Refresh Token 방식
Spring Security - 필터 체인 기반 보안 설정 (로컬/운영 분리)
Swagger - API 문서화 (에러 응답 커스텀 어노테이션 포함)
H2 Database - 로컬 개발용 인메모리 DB
레이어드 아키텍처 - Controller → Service → Repository 패턴
